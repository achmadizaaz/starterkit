## Git Rules

Selalu ikuti COMMIT_INSTRUCTIONS.md.

Setelah perubahan selesai dan stabil:

1. Review file yang berubah.
2. Kelompokkan perubahan berdasarkan modul.
3. Buat commit menggunakan Conventional Commit.
4. Jangan membuat commit generik seperti:
   - update
   - fix bug
   - revisi
   - perbaikan

Commit harus spesifik terhadap modul dan tujuan perubahan.
